# Marketplace immo Front

## Local development

**Requirements**

- Node.js version >= 14

**Development**

````
yarn
````

````
yarn run start
````

**Production**

````
yarn
````

````
yarn run build
````

## Note

Ensure the env variables are correctly declared in env.ts

## Technos

* React + Typescript
* Redux + React Thunk
* TailwindCSS + daisyUI
* Jest (Unit + integration testing)
* Cypress (E2E testing)
